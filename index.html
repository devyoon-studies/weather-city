서울 날씨 앱 - 웹 페이지

빈 페이지

간단한 빈 페이지입니다. WebView 테스트나 준비 중인 기능을 위해 사용할 수 있습니다.

도시 날씨 페이지

서울 각 구의 상세 날씨 정보를 보여주는 페이지입니다. URL 파라미터로 도시를 지정할 수 있습니다.

서울 구별 날씨 페이지

강남구 종로구 중구 용산구 성동구
React Native WebView에서 사용하기:
• 빈 페이지: file:///path/to/web/pages/empty.html
• 도시 페이지: file:///path/to/web/pages/city.html?city=강남구

JavaScript 인터페이스:
• window.updateWeatherData(data): 날씨 데이터 업데이트
• window.updateCityInfo(name, nameEn): 도시 정보 업데이트
